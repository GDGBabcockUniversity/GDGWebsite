# Team Member Images

This folder is for storing team member profile images.

## Image Guidelines

- **Format**: PNG or JPG
- **Recommended size**: 400x400px or larger (square aspect ratio preferred)
- **File naming**: Use lowercase with hyphens, e.g., `chukwuneku-akpotohwo.jpg`
- **Quality**: High-resolution images work best

## Folder Structure

\`\`\`
public/team/
├── core-team/          # Organizer, Co-Organizer, Technical Leads, Operations Leads, Media Leads
├── track-leads/        # Software, Data & AI, Infrastructure, Design & Management Track Leads
├── dev-team/           # Lead Backend/Frontend Developers, Backend Developers
└── specialists/        # All specialist roles
\`\`\`

## Usage in Code

To use an image in the team page, reference it like this:

\`\`\`tsx
image: "/team/core-team/chukwuneku-akpotohwo.jpg"
\`\`\`
